# 🏛️ Heritage Tourism Club - Server

This is the Express backend for the Heritage Tourism Club Management System.

For full project documentation, including setup instructions for both client and server, please refer to the [Root README](../README.md).

## Quick Start (Server)

1. Install dependencies:
   ```bash
   npm install
   ```

2. Setup environment variables in `.env`.

3. Run development server:
   ```bash
   npm run dev
   ```

4. Build for production:
   ```bash
   npm run build
   npm start
   ```
